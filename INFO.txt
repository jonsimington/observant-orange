Game Project #1
Chess game with random moves

Compiling and Running (on linux/mac):
----------------------------------------
"make" to compile
"./testRun SessionName" will run a game with name SessionName for another AI or person to join

Output:
---------------------------------------
The output will first display the move number being made that turn.  Then it will display the piece that is going to be moved and it's current location.  After that, all possible moves for that piece will be printed out along with the actual move made.
