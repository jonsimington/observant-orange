Game Project #2

Compiling and Running (on linux/mac):
----------------------------------------
"make" to compile
"./testRun SessionName" will run a game with name SessionName for another AI or person to join


Information:
---------------------------------------
There will be no output, game will be played using iterative deepening depth-limited minimax search by switching between white (high scores) and black (low scores) and searching up to 4 moves into the future OR until a score change is found.
